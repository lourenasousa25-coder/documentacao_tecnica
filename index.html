Introducao a Linguagem C

A linguagem C foi criada por Dennis Ritchie nos laboratórios da Bell Labs entre 1969 e 1973. É uma linguagem de programação de propósito geral, estruturada e de médio nível, que combina características de linguagens de alto e baixo nível. Até hoje é amplamente utilizada em sistemas operacionais, drivers, sistemas embarcados e softwares que exigem alto desempenho.

Todo programa em C começa sua execução pela função main(). A estrutura básica inclui a inclusão de bibliotecas com #include, declaração de variáveis, lógica do programa e retorno ao sistema operacional. O exemplo abaixo mostra o clássico "Olá, Mundo":

  • Criada em 1972 por Dennis Ritchie na Bell Labs
  • Base para linguagens como C++, Java e C#
  • Utilizada no desenvolvimento do sistema operacional Unix
  • Permite manipulação direta de memória com ponteiros
  • Padronizada pela ISO como C89, C99, C11 e C17
#include <stdio.h> int main() { printf("Ola, Mundo!\n"); return 0; }
Tipos de Dados e Variaveis

Em C, todas as variáveis devem ser declaradas antes de serem usadas, informando seu tipo. O tipo determina quanto espaço de memória será reservado e quais operações podem ser realizadas. Os tipos primitivos mais comuns são int, float, double, char e void.

C também permite a criação de tipos compostos como arrays, structs e unions. A escolha correta do tipo de dado é fundamental para o desempenho e a precisão do programa. O exemplo abaixo demonstra a declaração e uso dos principais tipos:

  • int — armazena números inteiros, geralmente com 4 bytes
  • float — armazena números reais com precisão simples (4 bytes)
  • double — armazena números reais com precisão dupla (8 bytes)
  • char — armazena um único caractere (1 byte)
  • unsigned — modificador que elimina o sinal, dobrando o valor positivo máximo
#include <stdio.h> int main() { int idade = 20; float altura = 1.75; double saldo = 1500.99; char inicial = 'A'; printf("Idade: %d\n", idade); printf("Altura: %.2f\n", altura); printf("Saldo: %.2lf\n", saldo); printf("Inicial: %c\n", inicial); return 0; }
Estruturas de Controle

As estruturas de controle determinam o fluxo de execução de um programa. Em C, as principais são as condicionais (if, else, switch) e os laços de repetição (for, while, do-while). Elas permitem que o programa tome decisões e repita blocos de código conforme necessário.

O uso correto das estruturas de controle é essencial para criar algoritmos eficientes. O laço for é ideal quando se sabe o número de repetições; o while é usado quando a condição de parada depende de um evento durante a execução. Veja abaixo exemplos de cada estrutura:

  • if/else — executa um bloco se a condição for verdadeira, outro caso contrário
  • switch — compara uma variável com múltiplos valores possíveis
  • for — repete um bloco um número definido de vezes
  • while — repete enquanto a condição for verdadeira
  • do-while — executa ao menos uma vez antes de verificar a condição
#include <stdio.h> int main() { int i; /* laco for: imprime de 1 a 5 */ for (i = 1; i <= 5; i++) { printf("%d ", i); } printf("\n"); /* condicional if/else */ if (i > 5) { printf("Loop encerrado.\n"); } else { printf("Ainda em execucao.\n"); } return 0; } #include <stdio.h> int main() { int nota = 7; switch (nota) { case 10: printf("Excelente!\n"); break; case 7: case 8: printf("Bom!\n"); break; case 5: case 6: printf("Regular.\n"); break; default: printf("Reprovado.\n"); } return 0; }
Funcoes

Funções são blocos de código reutilizáveis que realizam uma tarefa específica. Em C, toda função deve ser declarada com um tipo de retorno, um nome e uma lista de parâmetros. O uso de funções melhora a organização, legibilidade e manutenção do código, além de evitar repetição desnecessária.

C permite funções recursivas, ou seja, funções que chamam a si mesmas. Também é possível passar variáveis por valor ou por referência (usando ponteiros). A declaração antecipada de uma função é chamada de protótipo e deve aparecer antes de sua chamada no código.

A biblioteca padrão de C oferece diversas funções prontas, como printf(), scanf(), strlen() e malloc(), acessíveis via #include dos cabeçalhos correspondentes.

  • void — tipo de retorno para funções que não retornam valor
  • return — encerra a função e devolve um valor ao chamador
  • protótipo — declaração antecipada da assinatura da função
  • recursão — técnica onde a função chama a si mesma
  • parâmetros — valores recebidos pela função para processar
#include <stdio.h> /* prototipo da funcao */ int soma(int a, int b); int main() { int resultado = soma(3, 7); printf("Soma: %d\n", resultado); return 0; } int soma(int a, int b) { return a + b; } #include <stdio.h> /* funcao recursiva: fatorial */ int fatorial(int n) { if (n == 0) return 1; return n * fatorial(n - 1); } int main() { printf("5! = %d\n", fatorial(5)); return 0; }
Ponteiros e Memoria

Ponteiros são variáveis que armazenam o endereço de memória de outra variável. São um dos recursos mais poderosos — e perigosos — da linguagem C. Com ponteiros é possível manipular diretamente a memória, criar estruturas de dados dinâmicas como listas encadeadas e passar grandes volumes de dados para funções sem copiá-los.

A alocação dinâmica de memória em C é feita com as funções malloc(), calloc() e realloc(), todas da biblioteca stdlib.h. É obrigatório liberar a memória alocada com free() ao fim do uso, evitando vazamentos de memória (memory leaks).

  • & — operador que retorna o endereço de uma variável
  • * — operador de indireção, acessa o valor no endereço apontado
  • malloc() — aloca um bloco de memória sem inicializar
  • calloc() — aloca e inicializa a memória com zero
  • free() — libera a memória alocada dinamicamente
#include <stdio.h> int main() { int x = 10; int *ptr = &x; printf("Valor de x: %d\n", x); printf("Endereco de x: %p\n", ptr); printf("Valor via ponteiro: %d\n", *ptr); *ptr = 20; printf("Novo valor de x: %d\n", x); return 0; } #include <stdio.h> #include <stdlib.h> int main() { int n = 5; int *vetor = (int*) malloc(n * sizeof(int)); if (vetor == NULL) { printf("Erro ao alocar memoria.\n"); return 1; } for (int i = 0; i < n; i++) { vetor[i] = i * 2; printf("vetor[%d] = %d\n", i, vetor[i]); } free(vetor); return 0; }
